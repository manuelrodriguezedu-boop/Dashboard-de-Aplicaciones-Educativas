# **Propuesta de Intervención Educativa: Curso 2026-2027**

## ***Objetivo General:** Potenciar el razonamiento aplicado, la autonomía y la competencia digital mediante metodologías activas (Flipped Classroom) y el uso centralizado de plataformas educativas.*

\--------------------------------------------------------------------------------

# **Actuación 1: Razonamiento lógico y autocrítica en la resolución de problemas**

El propósito es implementar un programa que enseñe a los estudiantes a descomponer problemas complejos en pasos manejables y a identificar sus propios fallos de lógica.

* **Aplicación de métodos:** Se espera que el alumnado utilice sistemáticamente el desglose de datos, incógnitas y pasos en la mayoría de sus tareas de resolución.  
* **Identificación de errores:** Los estudiantes deben demostrar que son capaces de darse cuenta de dónde y por qué se han equivocado al pensar un problema. Esto debe reflejarse en una mejora de sus notas en los apartados de razonamiento de los exámenes.  
* **Contextualización:** Se utilizarán ejemplos de la vida diaria, como el manejo de presupuestos y dinero, para motivar al alumnado y dar sentido práctico al razonamiento.

| *Indicadores* |  |  |  |
| :---- | :---- | :---- | :---- |
| 1\. El alumnado aplica el desglose sistemático (datos, incógnitas y pasos) en la mayoría de sus tareas de resolución. |  |  |  |
| 2\. Los estudiantes demuestran que se dan cuenta de dónde y por qué se han equivocado al pensar un problema, mejorando su nota en el apartado de razonamiento de los exámenes. |  |  |  |
| 3\. Uso de ejemplos vinculados a la vida real (presupuestos, dinero) para aumentar la motivación y contextualizar el razonamiento. |  |  |  |

# **Actuación 2: Digitalización y Metodologías Activas (Moodle/Classroom)**

Esta línea busca centralizar la gestión de tareas y aplicar la "clase invertida puntual" para que el alumnado tenga un papel más participativo en el aula.

* **Plataformas digitales:** Se utilizarán **Classroom y Moodle** como centros operativos únicos para la entrega del 100% de las tareas y el acceso a materiales de apoyo audiovisual.  
* **Clase invertida (*Flipped Classroom*):** En temas seleccionados y puntualmente, los alumnos verán explicaciones en vídeo antes de la clase. De este modo, el tiempo en el aula se dedicará íntegramente a resolver dudas y realizar ejercicios de forma activa  
  * Nota: “**¿Qué son las "clases invertidas puntuales"?**Esta metodología (también llamada *flipped classroom*) consiste en darle la vuelta a la clase tradicional:  
    **Cómo funciona:** En lugar de que el profesor explique la teoría por primera vez en clase, el alumno **mira un vídeo corto o lee un material en casa antes de ir al aula**.

    **Para qué sirve en clase:** Como el alumno ya trae la idea "leída", el tiempo en el aula no se gasta en una charla del profesor, sino en **hacer ejercicios, resolver dudas y trabajar en grupo** mientras el docente guía el proceso.

    **Por qué "puntuales":** Significa que no se hará todos los días para no saturarlos, sino en temas específicos donde los alumnos suelen estar más "despistados" o pasivos, para obligarles a participar más desde el principio.”

* **Trabajo colaborativo:** Se fomentará el aprendizaje en equipo mediante la asignación de roles específicos (coordinador, secretario, técnico) o el liderazgo compartido en grupos pequeños para abordar proyectos de forma autónoma.

| *Indicadores* |  |  |  |
| :---- | :---- | :---- | :---- |
| 1\. Entrega del 100% de las tareas y acceso a materiales de apoyo a través de las plataformas centralizadas **Classroom y Moodle**. |  |  |  |
| 2\. Realización de sesiones de "clase invertida puntual" en temas seleccionados, usando vídeos tutoriales previos para que el tiempo en el aula se dedique a ejercicios y dudas. |  |  |  |
| 3\. Implementación de trabajo colaborativo con roles específicos (coordinador, secretario, etc.) o liderazgo compartido en grupos reducidos. |  |  |  |


# **Actuación 3: Atención a la Diversidad y Seguimiento Continuo**

Se enfoca en ofrecer un refuerzo personalizado y aumentar la frecuencia de las evaluaciones para fomentar un hábito de estudio constante.

* **Materiales personalizados:** Se emplearán cuadernillos de diseño propio adaptados específicamente para el alumnado con dificultades persistentes o de incorporación tardía.  
* **Evaluación frecuente:** Se realizarán pruebas parciales periódicas para evitar que la materia se acumule y para mantener la implicación de los estudiantes durante todo el trimestre.  
* **Coordinación bilingüe:** Se registrarán sistemáticamente las necesidades lingüísticas del alumnado a través de la plantilla de coordinación entre el profesorado de inglés y el de áreas no lingüísticas.

| *Indicadores* |  |  |  |
| :---- | :---- | :---- | :---- |
| 1\. Uso de cuadernillos de diseño propio adaptados para alumnos con dificultades persistentes o incorporación tardía. |  |  |  |
| 2\. Realización periódica de pruebas parciales para motivar la implicación continua y evitar la acumulación de materia. |  |  |  |
| 3\. Registro sistemático de necesidades lingüísticas en la plantilla de coordinación bilingüe (ANL/Inglés).Este indicador de logro consiste en la **estandarización y el uso regular de un documento compartido** (la plantilla de coordinación) entre los profesores de inglés y los de materias técnicas o científicas (Áreas No Lingüísticas o ANL, como Matemáticas o Tecnología). Sus objetivos principales son: **Unificar el registro:** Asegurar que todos los docentes que intervienen en el programa bilingüe utilicen el mismo formato para anotar qué dificultades o necesidades de idioma detectan en cada alumno. **Coordinación docente:** Facilitar que el profesor de inglés sepa exactamente qué vocabulario o estructuras lingüísticas necesitan reforzar los alumnos para entender las clases de ciencias o tecnología, y viceversa. **Atención personalizada:** Al tener este registro "sistemático" (es decir, constante y organizado), se puede realizar un seguimiento más preciso de los alumnos que tienen más dificultades con el idioma dentro de las materias técnicas, considerándolo una parte fundamental de la **atención a la diversidad**. En resumen, se trata de que no haya información dispersa, sino que exista un **documento único de trabajo** donde se vuelque todo lo que el alumnado necesita mejorar a nivel lingüístico para tener éxito en las áreas no lingüísticas. |  |  |  |

# **Claves Estratégicas para el Éxito del Curso:**

* **Entender el proceso mental:** El objetivo es que el alumno no solo llegue a la solución, sino que comprenda su propio razonamiento e identifique sus fallos lógicos de forma sencilla para mejorar sus resultados académicos.  
* **Docente como guía:** En el modelo de clase invertida y en el ámbito científico-tecnológico, el profesor actúa como guía del proceso, apoyándose constantemente en recursos audiovisuales para involucrar a los alumnos más pasivos.  
* **Responsabilidad digital:** Al tener toda la información y tareas centralizadas en plataformas accesibles desde cualquier dispositivo, se fomenta una mayor responsabilidad y autonomía en el estudiante.  
* **Hábitos de estudio:** Se reforzará el uso de la agenda escolar y técnicas como esquemas y resúmenes de manera interdisciplinar para asegurar una transición exitosa entre etapas.

# **Preguntas Frecuentes FAQ**

## ***¿Cómo se coordinarán Classroom y Moodle en el curso?***

La coordinación de Classroom y Moodle para el curso 2026-27 se basa en un modelo de dualidad digital diseñado para centralizar la gestión académica y fomentar la autonomía del alumnado.  
Según las fuentes, la integración de ambas plataformas se llevará a cabo de la siguiente manera:

* Centralización de tareas: Se utilizarán una de las plataformas como el canal único para la entrega del 100% de las tareas, asegurando que el alumnado tenga un lugar unificado para organizar sus deberes.  
* Acceso a materiales de apoyo: Tanto Classroom como Moodle servirán para alojar materiales audiovisuales y vídeos tutoriales, permitiendo que el alumnado acceda a los contenidos de ayuda desde cualquier dispositivo.  
* Gestión de la información: El objetivo de coordinar estas herramientas es garantizar que toda la información relevante esté en un solo lugar, lo que facilita el seguimiento por parte del estudiante y fomenta su responsabilidad individual.  
* Soporte para metodologías activas: Esta infraestructura digital será el pilar para implementar estrategias como la clase invertida (flipped classroom), donde el material teórico se revisa previamente en estas plataformas para aprovechar el tiempo en el aula en actividades prácticas y resolución de dudas.

# **¿Qué temas se elegirán para las clases invertidas puntuales?**

Según las fuentes proporcionadas, los temas para las clases invertidas puntuales se seleccionarán bajo los siguientes criterios y áreas específicas:

* Bloque de Álgebra: Se menciona como un área prioritaria para enseñar a descomponer problemas abstractos en pasos manejables.  
* Temas que generen pasividad: La metodología se aplicará en momentos específicos con el objetivo de involucrar al alumnado más pasivo, transformando el tiempo de clase en una sesión activa de resolución de dudas y ejercicios.  
* Contenidos que requieran apoyo audiovisual constante: Especialmente en el Ámbito Científico-Tecnológico, se elegirán temas donde el docente pueda actuar como guía del proceso, apoyándose en vídeos tutoriales para que el alumno revise la teoría previamente.  
* Temas de mayor dificultad conceptual: Se seleccionarán aquellos contenidos donde los estudiantes suelen estar más "despistados", permitiendo que traigan la idea leída de casa para practicar de forma más intensiva en el aula.

En resumen, aunque no se detalla un listado de unidades didácticas concretas, las fuentes señalan al álgebra y a aquellos contenidos que presentan mayores retos de participación o abstracción como los principales candidatos para esta metodología

# **¿Cómo se evaluará si los alumnos identifican sus errores lógicos?**

La evaluación de si los alumnos identifican sus errores lógicos se realizará a través de varios mecanismos complementarios, centrados tanto en los resultados académicos como en la observación directa del proceso de pensamiento:

* Análisis de los ítems de razonamiento en los exámenes: Se observará una mejora en las notas específicamente en los apartados de razonamiento de las pruebas escritas. El objetivo es que los estudiantes demuestren que comprenden dónde y por qué se han equivocado en su planteamiento inicial.  
* Informes y autoevaluaciones del departamento: Los docentes reportarán en las autoevaluaciones trimestrales si el alumnado evalúa mejor las alternativas existentes antes de actuar y si son capaces de detectar fallos en ejercicios de lógica.  
* Observación del "desglose sistemático": Se evaluará si los alumnos aplican correctamente la técnica de separar datos e incógnitas y los pasos lógicos, especialmente en el bloque de álgebra, lo que facilita ver en qué punto exacto del proceso ocurre el error.  
* Participación en la resolución activa: En las sesiones de "clase invertida" y puestas en común, el docente actuará como guía para verificar que el alumno entiende su propio proceso mental y no solo busca llegar a la solución final.

En resumen, la evaluación no se limitará a saber si la respuesta es correcta, sino que buscará evidencias de autocrítica y comprensión del proceso mental del estudiante

# **¿Qué ejemplos cotidianos se usarán para el razonamiento matemático?**

Para fomentar el razonamiento matemático y aumentar la motivación del alumnado, las fuentes proponen el uso de ejemplos vinculados directamente a la vida real y a los intereses cotidianos de los estudiantes.  
Los ejemplos específicos que se utilizarán incluyen:

* Manejo del dinero y presupuestos: Se emplearán problemas relacionados con la gestión económica personal para contextualizar el razonamiento, especialmente en grupos con menor implicación o en 2º de ESO.  
* Contextos de laboratorio: Para el alumnado de 4º de ESO, se propone aplicar conceptos matemáticos a través de proyectos interdisciplinares con las asignaturas de Física y Química, utilizando el entorno del laboratorio como escenario de aplicación práctica.  
* Situaciones de la vida cotidiana: De manera general, la propuesta para el curso 2026-2027 busca que el razonamiento lógico se trabaje mediante situaciones reales que ayuden a los estudiantes a dar un sentido práctico a la materia.
